# OpenAI Provider

Use ArtemisKit with OpenAI's GPT models.

## Setup

### 1. Get API Key

Get your API key from the [OpenAI Dashboard](https://platform.openai.com/api-keys).

### 2. Set Environment Variable

```bash
export OPENAI_API_KEY="sk-..."
```

### 3. Configure Scenario

```yaml
name: my-test
provider: openai
model: gpt-5.1

cases:
  - id: example
    prompt: "Hello"
    expected:
      type: contains
      values: ["hello"]
      mode: any
```

## Available Models

| Model | Description |
|-------|-------------|
| `gpt-5.1` | Latest flagship model |
| `gpt-5.1-mini` | Smaller, faster variant |
| `gpt-4o` | Previous generation multimodal |
| `gpt-4o-mini` | Fast and cost-effective |

## Config File

Configure OpenAI in your `artemis.config.yaml`:

```yaml
provider: openai
model: gpt-5.1

providers:
  openai:
    apiKey: ${OPENAI_API_KEY}
    organization: ${OPENAI_ORG_ID}  # Optional
    timeout: 60000
    maxRetries: 2
```

## Scenario-Level Options

Override settings in your scenario file:

```yaml
name: my-test
provider: openai
model: gpt-5.1
temperature: 0
seed: 42
maxTokens: 1000

cases:
  - id: example
    prompt: "Hello"
    expected:
      type: contains
      values: ["hello"]
      mode: any
```

## Organization ID

If you belong to multiple organizations:

```bash
export OPENAI_ORG_ID="org-..."
```

Or in your config:

```yaml
providers:
  openai:
    apiKey: ${OPENAI_API_KEY}
    organization: ${OPENAI_ORG_ID}
```
