# Anthropic Provider

Use ArtemisKit with Anthropic's Claude models.

## Setup

### 1. Get API Key

Get your API key from the [Anthropic Console](https://console.anthropic.com/).

### 2. Set Environment Variable

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

### 3. Configure Scenario

```yaml
name: my-test
provider: anthropic
model: claude-3-5-sonnet-20241022

cases:
  - id: example
    prompt: "Hello"
    expected:
      type: contains
      values: ["hello"]
      mode: any
```

## Available Models

| Model | Description |
|-------|-------------|
| `claude-sonnet-4-20250514` | Default model, balanced performance |
| `claude-sonnet-4-5-20250929` | Best balance of intelligence and speed |
| `claude-opus-4-5-20251101` | Most intelligent model |
| `claude-3-5-sonnet-20241022` | Legacy Sonnet (still supported) |
| `claude-3-5-haiku-20241022` | Fast and cost-effective |

## Config File

Configure Anthropic in your `artemis.config.yaml`:

```yaml
provider: anthropic
model: claude-3-5-sonnet-20241022

providers:
  anthropic:
    apiKey: ${ANTHROPIC_API_KEY}
    timeout: 60000
    maxRetries: 2
```

## Scenario-Level Options

Override settings in your scenario file:

```yaml
name: my-test
provider: anthropic
model: claude-3-5-sonnet-20241022
temperature: 0
maxTokens: 4096

cases:
  - id: example
    prompt: "Hello"
    expected:
      type: contains
      values: ["hello"]
      mode: any
```
