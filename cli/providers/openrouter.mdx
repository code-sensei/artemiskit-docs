# OpenAI Provider

Use ArtemisKit with OpenAI's GPT models.

## Setup

### 1. Get API Key

Get your API key from the [OpenAI Dashboard](https://platform.openai.com/api-keys).

### 2. Set Environment Variable

```bash
export OPENAI_API_KEY="sk-..."
```

### 3. Configure Scenario

```yaml
config:
  provider: openai
  model: gpt-4
```

## Available Models

| Model | Description |
|-------|-------------|
| `gpt-4` | Most capable GPT-4 model |
| `gpt-4-turbo` | GPT-4 Turbo with 128k context |
| `gpt-4o` | Optimized GPT-4 |
| `gpt-3.5-turbo` | Fast and cost-effective |

## Configuration Options

```yaml
config:
  provider: openai
  model: gpt-4
  temperature: 0        # 0-2, lower = more deterministic
  max_tokens: 1000      # Maximum response length
  top_p: 1              # Nucleus sampling
  frequency_penalty: 0  # Reduce repetition
  presence_penalty: 0   # Encourage new topics
```

## Organization ID

If you belong to multiple organizations:

```bash
export OPENAI_ORG_ID="org-..."
```
