# artemiskit compare

Compare metrics between two evaluation runs to detect regressions.

## Synopsis

```bash
artemiskit compare <baseline> <current> [options]
akit compare <baseline> <current> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `baseline` | Run ID or path to baseline run_manifest.json |
| `current` | Run ID or path to current run_manifest.json |

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `--threshold` | Regression threshold (0-1) | `0.05` (5%) |
| `--config` | Path to config file | `artemis.config.yaml` |

## Examples

### Basic Comparison

Compare two run manifests:

```bash
akit compare ./baseline/run_manifest.json ./current/run_manifest.json
```

### With Custom Threshold

Allow up to 10% regression before failing:

```bash
akit compare baseline.json current.json --threshold 0.1
```

### Compare by Run ID

If runs are saved to storage:

```bash
akit compare run_abc123 run_def456
```

## Output

The compare command outputs a detailed comparison:

```
Comparison: baseline vs current
===============================

| Metric       | Baseline | Current | Delta   | Status |
|--------------|----------|---------|---------|--------|
| Success Rate | 95.0%    | 92.0%   | -3.0%   | ✓      |
| Avg Latency  | 200ms    | 220ms   | +10.0%  | ⚠      |
| Pass Count   | 19/20    | 18/20   | -1      | ✓      |

Test Case Changes:
  ✗ greeting-test: PASS → FAIL
  
Summary: 1 regression detected (within threshold)
```

## Exit Codes

| Code | Meaning |
|------|---------|
| 0 | No significant regressions (within threshold) |
| 1 | Regressions detected (above threshold) |
| 2 | Error during comparison |

## CI/CD Integration

Use in GitHub Actions to gate deployments:

```yaml
- name: Run evaluation
  run: akit run scenarios/qa.yaml --save -o ./current

- name: Compare with baseline
  run: akit compare ./baseline/run_manifest.json ./current/run_manifest.json --threshold 0.05
```

## Baseline Strategies

Configure baseline selection in your config:

```yaml
# artemis.config.yaml
ci:
  failOnRegression: true
  regressionThreshold: 0.05
  baselineStrategy: latest  # 'latest', 'tagged', or 'specific'
  baselineRunId: run_abc123  # For 'specific' strategy
```
