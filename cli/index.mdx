# ArtemisKit CLI

The ArtemisKit CLI (`@artemiskit/cli`) is the primary way to run LLM evaluations.

## Installation

```bash
npm install -g @artemiskit/cli
```

## Commands

| Command | Description |
|---------|-------------|
| [`artemiskit run`](/docs/cli/commands/run) | Run scenario-based evaluations |
| [`artemiskit redteam`](/docs/cli/commands/redteam) | Security red team testing |
| [`artemiskit stress`](/docs/cli/commands/stress) | Load and stress testing |
| [`artemiskit compare`](/docs/cli/commands/compare) | Compare two evaluation runs |
| [`artemiskit report`](/docs/cli/commands/report) | Generate reports from saved runs |
| [`artemiskit history`](/docs/cli/commands/history) | View run history |
| [`artemiskit init`](/docs/cli/commands/init) | Initialize configuration |

You can use `akit` as a shorter alias for `artemiskit`.

## Quick Start

```bash
# Set your API key
export OPENAI_API_KEY="sk-..."

# Run a scenario (results are saved by default)
akit run scenarios/my-test.yaml

# Run without saving results
akit run scenarios/my-test.yaml --no-save
```

## Configuration

Create `artemis.config.yaml` for default settings:

```yaml
provider: openai
model: gpt-5

providers:
  openai:
    apiKey: ${OPENAI_API_KEY}
    timeout: 60000

output:
  format: json
  dir: ./artemis-output
```
