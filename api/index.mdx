# ArtemisKit Core API

<Aside type="caution" title="Coming Soon">
  The programmatic API (`@artemiskit/core`) is coming in v0.2.0. This documentation is a preview.
</Aside>

The ArtemisKit Core library allows you to embed LLM evaluation directly in your code.

## Use Cases

- **Test frameworks** — Integrate with Jest, Vitest, or Mocha
- **Custom tooling** — Build your own evaluation pipelines
- **Programmatic control** — Fine-grained configuration

## Preview

```typescript
import { Evaluator, OpenAIProvider } from '@artemiskit/core';

const evaluator = new Evaluator({
  provider: new OpenAIProvider({
    apiKey: process.env.OPENAI_API_KEY,
    model: 'gpt-4',
  }),
});

const result = await evaluator.run({
  prompt: 'What is 2+2?',
  expect: {
    contains: ['4'],
  },
});

console.log(result.passed); // true
```

## Jest Integration (Preview)

import { createMatcher } from '@artemiskit/jest';

const evaluateLLM = createMatcher({
